Build an OGRe mesh ready for export.

Remarks
Every PolygonMesh object is exported as a different SubMesh. Other object types are ignored.
Parameters
mergeSubMeshesWhether to merge submeshes with the same material
exportChildrenWhether to cascade down each objects children
edgeListsWhether to calculate edge lists
tangentsWhether to calculate tangents
animListList of animations in use (of any type)
materialPrefixPrefix to give all materials
lodLOD generation parameters (if required)
skeletonNameName of the skeleton to link to if animated
Returns
List of deformers (bones) which were found whilst exporting (if skeletonName was provided) which can be used to determine the skeleton.
